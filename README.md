# Mini CRM

A full-stack Mini CRM application built with Django REST Framework, React, and MySQL.

## Tech Stack

### Backend
- Django
- Django REST Framework
- MySQL
- Token Authentication

### Frontend
- React 
- Axios
- Recharts

## Features
- Email login
- Customer CRUD
- Soft delete
- Customer notes
- Opportunities management
- Dashboard with KPI cards and chart
